# Storytelling Framework Materials

This folder contains the theoretical source materials behind MANOE's storytelling system.

## Files

- `Storyteller full.pdf` - Full framework manual covering the narrative creation process.
- `Framework structure.pdf` - Visual workflow structure for the framework stages.
- `Storytelling Framework.docx` - Expanded "Storytelling Alchemist: Unbound" manual.
- `storyteller_workflow.json` - Workflow representation of the Storyteller process.
- `storyteller_complete_workflow.json` - Complete workflow export.
- `Storyteller (2).json` - Additional workflow export/reference.

These files are documentation/reference assets, separate from the implementation code.
